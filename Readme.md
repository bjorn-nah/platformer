# Platformer

*Proof of concept of Snes platformer using SNDK*

## About this project:
The goal is not to make a full game or a commercial product but
* play with SNDK to show what's capable of
* learn how a plateformer engine works
* play a little with physicis an parameters to experiment gamplay feelings
* have fun

## Ressources:
* SNDK is a SDK for SNES with Higueul language: https://github.com/Kannagi/SNDK
* Grapichs assets by Pixel Frog: https://pixelfrog-assets.itch.io/pixel-adventure-1
* Sound fx: https://jalastram.itch.io/8-bit-jump-sound-effects

## Build the project:
I remove all the SNDK files on this repository (see .gitignore) so to build you sould add missing files from the SNDK Release.
* Extract all files from the SNDK archive in a clean directory
* Copy to the project directory following files:
	* higueulc
	* higueulc.exe
	* run.sh
	* SNDK (Whole directory)
	* SPC (Whole directory)
	* tools  (Whole directory - not mandatory)
	* translator (Whole directory - not mandatory)
